# 8-bit-LFSR-MISR
1. Use CircuitVerse to build an 8-bit Galois LFSR whose specification is as follows: a. The characteristic polynomial is 1 + x 4 + x 5 + x 6 + x 8 .  b. Any 8-bit seed can be loaded by setting a signal LoadSeed = 1. During normal operation of LFSR, LoadSeed=0. 2. Build an 8-bit MISR
